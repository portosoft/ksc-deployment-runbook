# Versionamento

Este projeto utiliza **Semantic Versioning (SemVer)**:

- **MAJOR**: Mudanças que quebram a compatibilidade do runbook (ex: mudança de SO base de Rocky 9 para Ubuntu).
- **MINOR**: Novas funcionalidades ou scripts (ex: adição de suporte a PostgreSQL Externo).
- **PATCH**: Correções de erros na documentação ou bugfixes em scripts existentes.

As tags de versão devem seguir o formato `vX.Y.Z`.

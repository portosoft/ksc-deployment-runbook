# 12 - FAQ

## Perguntas Frequentes

**1. Posso usar o PostgreSQL 15?**
Sim, mas o PostgreSQL 16 é a versão recomendada para o KSC 16 devido a melhorias de performance.

**2. O KSC suporta interface gráfica no Linux?**
O servidor é CLI, mas o console de administração é via Web (HTTPS).

**3. Como mudo a porta do Web Console?**
Edite o arquivo de configuração em `/etc/opt/kaspersky/ksc-web-console/setup.json`.
